1、创建文章 powershell
hugo new posts/hello-world.md
打开文件
把文件头部的 draft: true 改为 draft: false；

2、本地启动预览
hugo server -D

3、推送本地代码到GitHub
powershell:

git add .
git commit -m "初始化 Hugo 站点"
git remote add origin https://github.com/<用户名>/<仓库名>.git
git branch -M main
git push -u origin main
